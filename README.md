# 🎙️ Whisper_Transcription | Advanced Speech-to-Text System

An advanced AI-powered speech-to-text system built using OpenAI Whisper.  
This project supports multi-language transcription, speaker detection, and includes a GUI-based interface with user authentication.

## 🚀 Features

- 🎤 High-accuracy audio transcription using Whisper  
- 🌐 Multi-language support (English, Hindi, etc.)  
- 🧠 Speaker detection (voice-based & basic)  
- 🖥️ GUI interface with login & register system  
- 🔐 MySQL authentication (XAMPP)  
- 📁 Auto-generated transcript files  
- ⏱️ Timestamp-based transcription  
- 📊 Detailed transcription report  

---

## 🛠️ Tech Stack

- Python  
- OpenAI Whisper  
- Tkinter (GUI)  
- MySQL (Database)  
- Librosa + Scikit-learn (Speaker detection)  

---

## ▶️ How to Run

```bash
pip install openai-whisper
pip install mysql-connector-python
pip install numpy scipy scikit-learn librosa
pip install soundfile

python whisper_gui_mysql.py
